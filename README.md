# Pulse Deck

自作した音声ファイルを手動再生して楽しむ、スマホ対応の音反応ビジュアライザーです。依存パッケージはありません。

## 使い方

1. `index.html` をブラウザで開く、または静的ホスティングへ公開します。
2. **MP3を選ぶ**から音源を指定します。
3. **再生**を押すと、音声の周波数に合わせてビジュアライザーが動きます。
4. **RING / WAVE** でビジュアルの種類を切り替えられます。

音声はサーバーに送信されず、選択した端末のブラウザ内で再生・解析されます。

## GitHub Pagesへの公開

このフォルダをGitHubリポジトリのルートに置き、GitHubの **Settings → Pages** で `Deploy from a branch` を選びます。公開対象を `main` ブランチの `/(root)` に設定すれば公開できます。

## ファイル構成

- `index.html` — 画面構造
- `style.css` — シンセサイザー風の見た目とスマホ対応レイアウト
- `app.js` — 手動再生、音声解析、Canvas描画

対応入力：MP3、WAV、OGG、M4A（ブラウザが対応している形式に限ります）。
